# StarNotary Token DAPP

This is a simple DAPP based on an ERC-721 token where you can register a new star and name it.

You can also get a star name by token ID.

## Token Info

- Name: `StarNotary`
- Symbol: `SNT`

## Versions

- Truffle: `2.1.4`
- Solidity compiler: `0.8.17`
- OpenZeppelin: `4.8.1`
